# vue2-vuetify

## Проект создан для быстрого старта приложений на технологиях vue2 + vuetify

### Project setup

```
yarn
yarn prepare
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
